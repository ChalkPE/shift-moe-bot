# shift-moe-bot
[사용 예 - @shift_moe](https://twitter.com/shift_moe)

이 트위터 봇은 Node.js로 돌아갑니다. 멘션을 하면 한화로 바꿔 답해줍니다.
아직 코드가 정리되어 있지 않습니다.

## 사용법
프로젝트 루트에 [`.env`](https://github.com/motdotla/dotenv) 파일을 만들어서 아래 정보를 입력해주세요.

```env
BOT_SCREEN_NAME=

CONSUMER_KEY=
CONSUMER_SECRET=
ACCESS_TOKEN_KEY=
ACCESS_TOKEN_SECRET=
```
